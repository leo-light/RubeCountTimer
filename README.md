# ルベカウントタイマー ver1.4

このツールは、特定の秒数でループするカウントダウンタイマーです。

## 使用方法（実行ファイルをご使用の方）
1. `ルベカウントタイマー.exe` を起動します。
   - ※初回起動時に警告が出る場合は「詳細情報」→「実行」を押してください。
2. **START**: メインのカウントダウンを開始します（ショートカット: [F9] キー）。
3. **CircleD**: 特殊なカウントを開始します（ショートカット: [F8] キー）。
   - ※途中でスタートを押すと、通常のカウントに切り替わります。
4. **STOP**: タイマーを止めて、秒数を最初に戻します。
5. **プリセット (1 / 2 / 3)**: あらかじめ設定した時間に切り替えられます。

## 開発者向け（ソースコードから実行・編集する場合）

### 必要な環境
- Python 3.10 以上
- 以下のライブラリが必要です：
  ```bash
  pip install PyQt6 pynput Pillow
  ```

### 実行方法
```bash
python main.py
```

### ファイル構成
- `main.py`: プログラム本体
- `style.qss`: デザイン設定（CSS）
- `settings.json`: 秒数やホットキーの設定データ
- `sounds/`: カウントダウン音声フォルダ
- `icon.ico`: アプリのアイコン
- `create_icon.py`: アイコン生成用スクリプト

## 各種設定
- **右クリック（各ボタン）**: 時間の変更、キーの変更、ボタン名の変更ができます。
- **右クリック（ウィンドウの背景）**: 言語切り替え（日本語/英語）、ホットキー全体の有効・無効を切り替えられます。
- **タイム補正 (LATENCY ADJUST)**: 計測中にスライダーを動かすことで、現在の時間を微調整できます。
