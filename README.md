# ルベカウントタイマー ver1.5

このツールは、特定の秒数でループするカウントダウンタイマーです。

## 使用方法（実行ファイルをご使用の方）
1. `ルベカウントタイマー.exe` を起動します。
   - ※初回起動時に警告が出る場合は「詳細情報」→「実行」を押してください。
2. **START**: メインのカウントダウンを開始します（ショートカット: [F9] キー）。
3. **CircleC**: サークルC用の特殊カウントを開始します（ショートカット: [F8] キー）。
   - ※初動とループで個別の秒数を設定可能です。
4. **STOP**: タイマーを止めて、完全にリセットします。
5. **プリセット (1 / 2 / 3)**: あらかじめ設定した時間に切り替えられます。
   - **プリセット3の特殊仕様**: このプリセットのみ、5秒前警告が「赤」と「黄色」で交互に切り替わります。

## 各種設定
- **右クリック（各ボタン）**: 時間の変更、キーの変更、ラベル名の変更ができます。
- **右クリック（ウィンドウの背景）**: 言語切り替え（日本語/英語）、ホットキー全体の有効・無効を切り替えられます。
- **タイム補正 (LATENCY ADJUST)**: 計測中にスライダーを動かすことで、現在の時間をリアルタイムで微調整できます。
