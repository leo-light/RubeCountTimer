========================================================================
　ルベカウントタイマー ver1.5
========================================================================

このツールは、特定の秒数でループするカウントダウンタイマーです。


【最初にすること】
1. 「ルベカウントタイマー.exe」をダブルクリックして起動します。
   （※初回起動時に警告が出る場合は「詳細情報」→「実行」を押してください）

------------------------------------------------------------------------
■ 基本操作（ボタンの使い方）
------------------------------------------------------------------------

● スタート (START)
   - メインのカウントダウンを開始します。
   - キーボードの [F9] キーでも開始できます。

● サークルC (CircleC)
   - サークルC用の特殊なカウント（初回5秒、以後22.5秒ループなど）を開始します。
   - キーボードの [F8] キーでも開始できます。

● ストップ (STOP)
   - タイマーを止めて、完全に初期状態にリセットします。

● プリセット (1 / 2 / 3)
   - 画面上の「1」「2」「3」ボタンで、登録した時間に切り替えられます。
   - 【プリセット3】のみ、5秒前警告が「赤」と「黄色」で交互に鳴ります。

------------------------------------------------------------------------
■ 各種設定（右クリックで変更）
------------------------------------------------------------------------

● キーや時間を変えたいとき
   - 各ボタン（STARTや1, 2, 3など）を【右クリック】してください。
   - 以下のことが設定できます：
     ・好きなキーボードのキーに変更
     ・カウントダウンする秒数の変更（初回／ループ別設定可能）
     ・ラベル名の変更

● 言語や全体の有効・無効
   - ウィンドウの「背景部分」を【右クリック】して「Settings（設定）」を選びます。

● タイム補正 (LATENCY ADJUST)
   - 計測中にスライダを動かすと、リアルタイムで現在の時間を微調整できます。
